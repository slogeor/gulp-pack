## fepack

#### 安装说明

1. git clone https://github.com/slogeor/fepack.git
2. cd fepack
3. npm gulp -g
4. npm webpack -g
5. npm install

#### 开发环境

``` gulp ```

查看network，没有混淆压缩的

#### 线上环境

``` build ```

查看network，混淆且有版本信息
